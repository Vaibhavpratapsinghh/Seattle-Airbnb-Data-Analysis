# Seattle Airbnb Data Analysis Dashboard (Tableau)

## 📌 Project Overview
This project analyzes the Seattle Airbnb dataset to uncover pricing patterns, revenue trends, and location-based insights. An interactive Tableau dashboard was developed to enable intuitive data exploration and support data-driven decision-making.

---

## 📊 Dashboard Highlights
- 📍 **Geospatial Analysis**: Price variation across different zipcodes using map visualization  
- 🛏️ **Bedroom-Based Pricing**: Comparison of average prices by number of bedrooms  
- 📈 **Revenue Trends**: Yearly revenue patterns to identify seasonal demand fluctuations  

---

## 🛠️ Tools & Technologies
- **Tableau** – Data visualization & dashboard creation 
- **Dataset** – Seattle Airbnb Open Data (Kaggle)

---

## 📂 Dataset
- Listings data (property details, pricing, location)  
- Calendar data (availability, pricing over time)  
- Reviews data (customer feedback)

Source: https://www.kaggle.com/datasets/airbnb/seattle

---

## 🔍 Key Insights
- Significant **price variation across zipcodes**, indicating strong location impact  
- **1-bedroom listings contribute higher revenue**, suggesting demand concentration  
- Clear **seasonal trends in revenue**, useful for pricing and availability strategies
